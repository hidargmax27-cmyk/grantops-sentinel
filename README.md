# GrantOps Sentinel

GrantOps Sentinel is an open-source public proof and risk signal dashboard for Base and Arbitrum builders.

It helps early dApp teams show what they have shipped, where contracts are deployed, whether source code is verified, what privileged contract controls may exist, and what milestones or early onchain activity can be referenced in grant applications.

## What The MVP Does

- Register Base and Arbitrum projects.
- Create public project proof pages.
- Show contract addresses and explorer links.
- Track source verification status.
- Flag basic risk signals such as proxy, upgrade, pause, mint, blacklist, sweep, and withdraw permissions.
- Show milestone evidence and recent activity metrics.
- Export a JSON report for grant reviewers or internal tracking.

## Why This Matters

Grant reviewers and ecosystem teams often need to manually verify whether a project has actually shipped. Builders also need a clean way to prove their deployment, milestones, and early impact without scattering evidence across explorers, GitHub, social posts, and private docs.

GrantOps Sentinel gives builders a standardized proof page and gives reviewers a faster way to inspect shipped work.
