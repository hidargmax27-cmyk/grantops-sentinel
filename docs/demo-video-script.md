# Demo Video Script

Target length: 60-90 seconds.

## Script

Hi, I am Yutong Zhang, building Cognitive Bridge.

We are building GrantOps Sentinel, an open-source public proof and risk signal dashboard for Base and Arbitrum builders.

The problem is simple: early dApp teams often have scattered proof. Contract addresses are on explorers, code is on GitHub, milestones are in docs, updates are on social media, and grant reviewers need to manually piece everything together.

GrantOps Sentinel gives each project a clean public proof page.

In the demo, I can register a project, select Base or Arbitrum, add a contract address, link GitHub, add a milestone, and mark whether the source is verified or the contract uses a proxy.

The page then shows explorer links, source verification status, early activity metrics, and risk signals such as proxy, upgrade, pause, mint, blacklist, sweep, or withdraw permissions.

This is not an audit certificate. It is a public risk signal and evidence layer that helps builders prove shipped work and helps reviewers know what to inspect next.

Our first milestone is Base and Arbitrum support. Next, we will add automated explorer API data, 20 indexed ecosystem examples, and Solana program evidence support.

We are applying for ecosystem support to keep this open-source, improve automated risk signals, and make proof pages useful for more builders.
